1. Contexte
   
La cybersécurité est devenue un enjeu majeur pour les entreprises. Les organisations doivent
identifier leurs actifs numériques, évaluer leurs vulnérabilités et estimer leur niveau de risque
afin de mettre en place des mesures de protection adaptées.
L'objectif de ce projet est de développer une application web Fullstack permettant de modéliser
une entreprise fictive, de gérer ses actifs informatiques et d'évaluer automatiquement son niveau
de risque cyber.
